Anime Rating Prediction API

Get accurate anime ratings based on different parameters

How it Works

The API uses the information provided by the user to predict the rating of an anime. The following parameters are used to make the prediction:

  • title
  • genders
  • studio
  • producers
  • synopsis
  • type

To get a prediction, you can use a POST request to the endpoint / and include the information in the request body. The API will return the predicted rating in a JSON format.
